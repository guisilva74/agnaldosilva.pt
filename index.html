HTML

A internet utiliza uma linguagem que permite formatar as paginas webs que visitamos.

HTMl significa HyperText Markup Language e é uma linguagem utilizada para criar paginas com hipertexto (com hiperligações).

Os sinais de <> delimitam as tags (etiquetas) e o simbolo / permite fechar as tags.
